## 🟣 Tunnel
Expose your local server securely to the internet without opening router ports.
➡️ See the full guide here:
- [Tunnel](Tunnel.md)
Examples include:
- Secure exposure using Cloudflare Tunnel, Ngrok, or similar tools
- No need to configure port forwarding or a static IP
