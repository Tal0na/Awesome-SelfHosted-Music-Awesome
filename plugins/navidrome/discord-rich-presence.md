# 🎮 Discord Rich Presence Plugin

Repository: https://github.com/navidrome/discord-rich-presence-plugin

## Description

Share your listening activity directly on Discord with Rich Presence integration. The plugin displays the track you're currently playing, along with artist information, album artwork, and playback status, allowing friends and communities to see what you're listening to in real time.

## Features

* Real-time playback status
* Artist and track information
* Album artwork support
* Discord Rich Presence integration
* Automatic updates while listening
* Works with Navidrome playback sessions

## Installation

### Docker

1. Download the latest plugin release.
2. Place the plugin files in your Navidrome plugins directory.
3. Restart the Navidrome container.

### Example Configuration

Enable the plugin in your Navidrome plugins setup and follow the repository instructions to connect it with your Discord application.

## Compatibility

* Navidrome (Plugin System)
* Docker deployments
* Linux servers
* Discord Desktop Client
