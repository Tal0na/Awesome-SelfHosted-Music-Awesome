# 🧠 ListenBrainz Daily Playlist

Repository: https://github.com/kgarner7/navidrome-listenbrainz-daily-playlist

## Description

Automatically imports personalized daily playlists generated by ListenBrainz into Navidrome. The plugin helps users discover music based on their listening history and recommendations from the ListenBrainz ecosystem, making curated playlists available directly from their self-hosted music server.

## Features

* Daily playlist synchronization
* Personalized music recommendations
* ListenBrainz integration
* Automatic playlist updates
* Library discovery and exploration
* Seamless Navidrome integration

## Installation

### Docker

1. Download the latest plugin release.
2. Place the plugin files in your Navidrome plugins directory.
3. Restart the Navidrome container.

### Example Configuration

Enable ListenBrainz support:

```yaml
environment:
  ND_LISTENBRAINZ_ENABLED: "true"
  ND_LISTENBRAINZ_USERTOKEN: "YOUR_TOKEN_HERE"
```

Configure the plugin according to the repository documentation and ensure your ListenBrainz account is connected.

## Compatibility

* Navidrome (Plugin System)
* Docker deployments
* Linux servers
* ListenBrainz accounts
