# 🎧 Best MacOS Clients for Self-host Music

A curated list of recommended MacOS clients compatible with **Self-host Music** and other Subsonic-compatible servers.

---

## ⭐ Recommended Clients

### Feishin

- **API Support:** ✅ Navidrome / ✅ Jellyfin / ✅ OpenSubsonic
- **License:** GPL-3.0
- **Open Source:** ✅ Yes
- **Price:** 🆓 Free
- **Rating:** ⭐⭐⭐⭐⭐
- **Last Update:** 🟢 Actively maintained
- **Highlights:**
  - MPV player backend
  - Web player backend
  - Modern UI
  - Smart playlist editor (Navidrome)
  - Synchronized and unsynchronized lyrics support

### 🎧 kopuz

- **API Support:** ✅ Navidrome / ✅ Jellyfin
- **License:** MIT License
- **Open Source:** ✅ Yes
- **Price:** 🆓 Free
- **Status:** 🟢 Actively maintained
- **Rating:** ⭐⭐⭐⭐⭐

#### 🚀 Highlights
- Kopuz is a modern, lightweight, music player application built with Rust and the Dioxus framework. It provides a clean and responsive interface for managing and enjoying your local music collection.
